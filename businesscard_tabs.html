Artwork Setup

Please set up your artwork to the following specifications:

Size with Bleed: 61mm x 91mm
Resolution: 300 dpi minimum
File Format: PDF/X-1a (although we can accept other PDF formats)
Colour: CMYK (RGB, spot and Pantone colours will be converted)
Ink Coverage: 280% maximum for uncoated paper and 300% maximum for coated paper
Fonts: Outlined or Embedded
Quiet Zone: 5mm minimum
Bleed: 3mm minimum
Trim Marks: Yes
Transparencies: Flattened

Artwork Service

What is included in your 'Just Print' artwork service?

Just Print is our free artwork service, where we will 'Just Print' your print ready PDF/X-1a file, as supplied.

Please note: We only accept PDF/X-1a files with this service. If you have any other file format, you will need to select 'File Check with Proof'.

We recommend this service only to individuals who fully understand our print ready requirements. Before using this service, you should be confident that your PDF/X-1a files have been supplied correctly.

The following terms apply when using the 'Just Print' service:

  • Your order will be processed through our automated production system and rejected only if we are unable to print your file.
  • Files must be PDF/x-1a format.
  • We will not carry out any pre-flight or quality checks on your file as these will already have been carried out by yourself prior to submitting the file.
  • You will not receive a proof.
  • Booklets and folders are particularly complex and are generally higher value items, therefore we strongly recommend 'File Check with Proof' on these products.
  • When using this service to order booklets, artwork should be supplied as one multi-page PDF/X-1a containing single pages in the correct order.

We will automatically fix these errors within your file:

  • Convert RGB to CMYK.
  • Convert spot and Pantone colours to CMYK.
  • Missing bleed.
  • Ink coverage over 280% for uncoated paper and 300% for coated paper.
  • Overprints set on colours other than black.

We will notify you of a file rejection if your file contains any of these errors:

  • Corrupt PDF.
  • Un-embedded fonts.
  • Password protected.
  • Sizes that are disproportionate.
  • Incorrect number of files.

What is included in your 'File Check with Proof' artwork service?

'File Check with Proof' offers a safety net of a double check on your print ready file before proceeding to print.

We will accept a print ready PDF/X-1a, as well as print ready EPS, TIF or JPG files. We will assume your file has been designed and prepared correctly, adhering to our print ready requirements, and you can also choose to receive a proof for an additional fee.

We will automatically fix these errors within your file:

  • Convert JPG, TIF or EPS to PDF/X-1a.
  • Convert RGB to CMYK.
  • Convert spot and Pantone colours to CMYK.
  • Overprints set on colours other than black.
  • Ink coverage over 280% for uncoated paper and 300% for coated paper.
  • Missing bleed.

We will warn you if your file contains any of these errors:

  • Text/objects too close to the edge of the page.
  • Low resolution images.
  • Impure blacks.
  • Incorrect panel sizes on folded jobs.
  • Issues with the form cutter guide lining up with your artwork.
  • No allowance for creep within your booklet.

We will reject your file if it contains any of these errors:

  • Corrupt file.
  • Un-embedded fonts.
  • Password protected.
  • Sizes that are disproportionate.
  • Missing elements from the file.
  • Incorrect number of files.
